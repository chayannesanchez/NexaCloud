# Código de conducta

Este proyecto promueve una colaboración respetuosa, profesional e inclusiva.

## Comportamientos esperados

- Usar lenguaje respetuoso.
- Dar retroalimentación constructiva.
- Respetar diferentes niveles de experiencia.
- Explicar los cambios de forma clara.
- Evitar compartir información sensible.

## Comportamientos no aceptados

- Insultos o ataques personales.
- Publicar credenciales o datos privados.
- Comentarios discriminatorios.
- Cambios maliciosos o destructivos.

## Aplicación

Los mantenedores pueden cerrar issues, rechazar pull requests o restringir participación si se incumple este código de conducta.
